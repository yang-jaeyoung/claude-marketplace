# 주식 종합분석 리포트

종목의 종합분석 리포트를 생성합니다.

## 입력 파싱

1. **종목 식별**: 첫 번째 인자에서 종목명 또는 6자리 종목코드 추출
2. **저장 경로**: 두 번째 인자 또는 `--output` 플래그에서 경로 추출
3. **기본 경로**: 미지정시 콘솔 출력만 수행

## 분석 순서

### Step 1: 종목 확인
```
krx_collect(dataType: "tickers", market: "KOSPI")
krx_collect(dataType: "tickers", market: "KOSDAQ")
```
- 종목명/코드로 검색하여 정확한 종목코드 및 시장 확인

### Step 2: 데이터 수집
```
krx_collect(dataType: "ohlcv", ticker: "{코드}", startDate: "1년전", endDate: "오늘")
krx_collect(dataType: "fundamental", ticker: "{코드}")
krx_collect(dataType: "marketcap", market: "{시장}")
```

### Step 3: 팩터 분석
```
factor_analyze(ticker: "{코드}", factors: ["PER", "PBR", "MOM_1M", "MOM_3M", "MOM_6M", "ROE"])
```

### Step 4: 유사 종목 스크리닝
```
stock_screen(
  conditions: ["시총>{시총*0.5}", "시총<{시총*1.5}", "PER>{PER*0.7}", "PER<{PER*1.3}"],
  market: "{시장}",
  limit: 10
)
```

### Step 5: 리포트 생성

마크다운 형식의 종합 리포트 생성:

```markdown
# {종목명} ({종목코드}) 종합분석 리포트

생성일: {오늘날짜}

## 요약
- 현재가: {현재가}원 ({전일대비})
- 시가총액: {시총}억원
- PER: {PER} / PBR: {PBR}
- 팩터점수: {점수} (상위 {백분위}%)
- 투자의견: {의견}

## 1. 기본 정보
...

## 2. 가격 동향
...

## 3. 밸류에이션 분석
...

## 4. 팩터 분석
...

## 5. 유사 종목 비교
...

## 6. 매수 타이밍 분석
...

## 7. 투자 의견 요약
...
```

### Step 6: 저장 (경로 지정시)
- 디렉토리 없으면 생성
- 파일명: `{종목명}_{날짜}.md`
- 저장 완료 메시지 출력

## 사용 예시

```bash
# 기본 사용
/stock-report 동운아나텍
/stock-report 094170

# 저장 경로 지정
/stock-report 동운아나텍 report/
/stock-report 094170 --output ./analysis/dongwoon.md

# 자연어 (자동 라우팅)
"동운아나텍 종합분석해서 report/에 저장해줘"
```
