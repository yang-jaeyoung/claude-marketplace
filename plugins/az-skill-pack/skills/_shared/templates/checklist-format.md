# Checklist Format Template

공통 체크리스트 포맷입니다.

## 상태 아이콘

| 아이콘 | 의미 |
|--------|------|
| ✅ | 구현됨 |
| ⚠️ | 부분구현 |
| ❌ | 미구현 |
| 🔍 | 수동확인 필요 |
| ➖ | 해당없음 (N/A) |

## 심각도 레벨

| 레벨 | 아이콘 | 대응 기한 |
|------|--------|----------|
| Critical | 🔴 | 24시간 내 |
| High | 🟠 | 1주 내 |
| Medium | 🟡 | 1개월 내 |
| Low | 🟢 | 분기 내 |

## 결과 요약 테이블

```markdown
| Category | ✅ | ⚠️ | ❌ | Score |
|----------|---|---|---|-------|
```
