# Brainstorming Techniques

## 기본 기법

### 1. Divergent Expansion (기본)

세 가지 관점에서 아이디어 확장:

```
🎯 Direct Solutions - 직접적인 해결책
🔄 Alternative Approaches - 대안적 접근법
💡 Creative/Unconventional - 창의적/비전통적 아이디어
```

### 2. SCAMPER

기존 아이디어 변형 기법:

| Action | Question |
|--------|----------|
| **S**ubstitute | 대체할 수 있는 것은? |
| **C**ombine | 합칠 수 있는 것은? |
| **A**dapt | 다른 곳에서 가져올 것은? |
| **M**odify | 바꾸거나 확대/축소할 것은? |
| **P**ut to other uses | 다른 용도는? |
| **E**liminate | 제거할 것은? |
| **R**everse | 뒤집거나 재배열할 것은? |

### 3. Six Thinking Hats

다각적 사고 기법:

| Hat | Focus |
|-----|-------|
| ⚪ White | 사실 & 데이터 |
| 🔴 Red | 감정 & 직관 |
| ⚫ Black | 리스크 & 주의 |
| 🟡 Yellow | 이점 & 낙관 |
| 🟢 Green | 창의성 & 대안 |
| 🔵 Blue | 프로세스 & 요약 |

### 4. Rapid Fire

빠른 아이디어 생성:
- 15-20개 이상 아이디어 생성
- 생성 중 비판 금지
- 이전 아이디어에서 확장

## 고급 기법

### Reverse Brainstorming

문제 반전 기법:
1. 문제 정의
2. 반전: "어떻게 이 문제를 일으킬 수 있을까?"
3. 반전된 문제에 대한 아이디어 생성
4. 각 아이디어를 뒤집어 해결책 도출

### Starbursting

5W1H 질문 기반:

```
         WHO?
          |
  WHY? -- ⭐ -- WHAT?
        / | \
   WHEN? WHERE? HOW?
```

### Mind Mapping

중심 주제에서 방사형 확장:
- 핵심 개념을 중심에 배치
- 관련 아이디어를 가지로 연결
- 세부 아이디어를 하위 가지로 확장

### Attribute Listing

구성요소 분해 및 변형:

| Component | Current | Alternatives |
|-----------|---------|--------------|
| Material | | |
| Shape | | |
| Process | | |

### Role Storming

다양한 관점에서 브레인스토밍:

| Persona | Question |
|---------|----------|
| Expert | 전문가라면? |
| Beginner | 초보자가 보면? |
| Competitor | 경쟁자라면? |
| Customer | 고객이 원하는 것은? |

### Worst Possible Idea

의도적으로 나쁜 아이디어 생성:
1. 최악의 해결책 질문
2. 나쁜 아이디어 생성 (비판 없이)
3. 왜 나쁜지 분석
4. 인사이트를 뒤집어 좋은 해결책 도출

## 기법 조합

복잡한 문제에는 기법 체인:

1. **Divergent → SCAMPER**: 광범위한 아이디어 후 최적 변형
2. **Six Hats → Rapid Fire**: 구조화된 분석 후 창의적 확장
3. **Reverse → Mind Map**: 반패턴 파악 후 해결책 매핑
