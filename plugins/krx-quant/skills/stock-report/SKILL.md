# 주식 종합분석 리포트 스킬

한국 주식(KOSPI/KOSDAQ) 종목의 종합분석 리포트를 생성합니다.

## 자동 활성화 패턴

다음과 같은 자연어 요청에 이 스킬이 자동 활성화됩니다:

- "삼성전자 분석해줘"
- "005930 종합분석"
- "동운아나텍 매수타이밍 분석해서 저장해줘"
- "SK하이닉스 리포트 만들어줘"
- "카카오 투자분석 report/에 저장"

## 분석 워크플로우

### 1단계: 종목 식별
- 종목명 또는 6자리 종목코드로 종목 식별
- KOSPI/KOSDAQ 시장 자동 판별

### 2단계: 데이터 수집 (krx_collect)
```
1. 기본 정보: 종목코드, 종목명, 시장
2. 가격 데이터: 최근 1년 OHLCV
3. 펀더멘털: PER, PBR, EPS, BPS, 배당수익률
4. 시가총액
```

### 3단계: 팩터 분석 (factor_analyze)
```
1. 팩터 노출도: 가치/모멘텀/퀄리티/사이즈 전 팩터
2. 종합 점수 및 시장 내 백분위
3. 섹터 평균 대비 비교
```

### 4단계: 유사종목 스크리닝 (stock_screen)
```
1. 동일 시장 내 유사 밸류에이션 종목
2. 시총 ±50% 범위 내 종목
3. 동일 섹터 종목 (가능한 경우)
```

### 5단계: 매수 타이밍 분석
```
1. 현재 가격 vs 52주 고가/저가
2. 모멘텀 팩터 기반 추세 분석
3. 밸류에이션 팩터 기반 적정가 분석
4. 기술적 지지/저항선 (20일/60일/120일 이동평균)
```

### 6단계: 리포트 생성
```markdown
# [종목명] (종목코드) 종합분석 리포트

생성일: YYYY-MM-DD

## 1. 기본 정보
## 2. 가격 동향
## 3. 밸류에이션 분석
## 4. 팩터 분석
## 5. 유사 종목 비교
## 6. 매수 타이밍 분석
## 7. 투자 의견 요약
```

## 사용 예시

### 기본 분석
```
/stock-report 동운아나텍
/stock-report 094170
```

### 리포트 저장
```
/stock-report 동운아나텍 report/
/stock-report 094170 --output report/dongwoon.md
```

### 자연어 요청
```
"동운아나텍 종합분석해서 report/에 저장해줘"
"삼성전자 매수타이밍 분석 리포트 만들어줘"
"SK하이닉스 투자분석 보고서 작성해줘"
```

## MCP 도구 사용

이 스킬은 다음 MCP 도구들을 순차적으로 사용합니다:

1. **krx_collect** (dataType: tickers) - 종목 확인
2. **krx_collect** (dataType: ohlcv) - 가격 데이터
3. **krx_collect** (dataType: fundamental) - 펀더멘털
4. **krx_collect** (dataType: marketcap) - 시가총액
5. **factor_analyze** (ticker: 종목코드) - 팩터 노출도
6. **stock_screen** (conditions: 유사조건) - 유사종목

## 출력 형식

### 콘솔 출력 (요약)
```
📊 동운아나텍 (094170) 종합분석

현재가: 8,650원 (전일대비 +2.1%)
시가총액: 2,850억원 (KOSDAQ 상위 15%)

밸류에이션:
  PER 12.5 (섹터평균 18.3 대비 저평가)
  PBR 1.8 (섹터평균 2.5 대비 저평가)

팩터 점수: +0.62 (상위 27%)
  가치: +0.45 | 모멘텀: +1.05 | 퀄리티: +0.80

매수 타이밍:
  52주 고가 대비: -12% (매수 적정 구간)
  모멘텀: 상승 추세 유지
  지지선: 8,200원 (20일 이평선)

💡 투자의견: 매수 고려 (저평가 + 상승모멘텀)

📁 리포트 저장: report/동운아나텍_20250129.md
```

### 파일 출력 (상세 리포트)
지정된 경로에 마크다운 형식의 상세 리포트 저장
