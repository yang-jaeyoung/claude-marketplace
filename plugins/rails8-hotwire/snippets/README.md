# Snippets Directory

Code snippets are located in the knowledge base.

## Actual Location

All snippets are in: **`../knowledge/snippets/`**

### Structure:
```
knowledge/snippets/
├── common/         # Common Rails patterns
├── javascript/     # JavaScript & Stimulus snippets
├── ruby/          # Ruby & Rails snippets
└── views/         # ERB & view snippets
```

## Usage

Reference snippets from the knowledge base:
```
/Users/urd/.claude/plugins/rails8-hotwire/knowledge/snippets/
```

## Available Snippet Categories

1. **Common** (`knowledge/snippets/common/`)
   - Shared patterns across domains

2. **Ruby** (`knowledge/snippets/ruby/`)
   - Ruby code snippets
   - Rails patterns

3. **JavaScript** (`knowledge/snippets/javascript/`)
   - Stimulus controllers
   - Turbo interactions

4. **Views** (`knowledge/snippets/views/`)
   - ERB templates
   - Partial patterns
