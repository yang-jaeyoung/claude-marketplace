# rails8-models

ActiveRecord Model Skill

## Invocation
`/rails8:models`

## Description
Guides ActiveRecord models, associations, queries, and validations.

## Topics
- Model definition and associations (has_many, belongs_to, etc.)
- Validations
- Callbacks
- Scopes
- Enums
- Query optimization
- N+1 problem resolution

## Key Patterns
- Eager loading with `includes`
- Query objects for complex queries
- Concerns for shared behavior
- Soft delete implementation
