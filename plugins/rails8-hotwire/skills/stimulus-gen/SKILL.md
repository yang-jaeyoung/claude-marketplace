# stimulus-gen

Stimulus 컨트롤러 생성

## Invocation
`/rails8:stimulus-gen`

## Description
Stimulus 컨트롤러를 생성합니다.

## Usage
```
/rails8:stimulus-gen dropdown
/rails8:stimulus-gen modal --targets="content,backdrop" --values="open:Boolean"
```

## Generated
- app/javascript/controllers/{name}_controller.js
- 사용 예제 HTML
