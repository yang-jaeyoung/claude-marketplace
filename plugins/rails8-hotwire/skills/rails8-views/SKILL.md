# rails8-views

View Layer Skill

## Invocation
`/rails8:views`

## Description
Guides view templates, partials, components, and forms.

## Topics
- ERB templates
- Partials and collections
- ViewComponent
- Layouts
- Form builder (form_with)
- Helper methods
- Tailwind CSS integration

## Key Patterns
- Collection rendering with caching
- Nested forms (accepts_nested_attributes)
- Empty state handling
- Turbo Frame partials
