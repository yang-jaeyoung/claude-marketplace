# Common Snippets

공통으로 사용되는 보일러플레이트 코드 템플릿입니다.

## 파일 목록

| 파일 | 설명 | 사용처 |
|------|------|--------|
| `stimulus-base.js` | Stimulus 컨트롤러 기본 구조 | 모든 Stimulus 컨트롤러 |
| `rspec-setup.rb` | RSpec 테스트 설정 | spec/rails_helper.rb |
| `policy-base.rb` | Pundit 정책 기본 클래스 | app/policies/ |
| `turbo-frame.erb` | Turbo Frame 패턴 모음 | Views |
| `job-base.rb` | ActiveJob 기본 클래스 | app/jobs/ |
| `factory-base.rb` | FactoryBot 팩토리 패턴 | spec/factories/ |

## 관련 스니펫

서비스 객체 기본 클래스는 `../ruby/services/application_service.rb`를 참조하세요.

## 사용법

각 파일은 복사하여 프로젝트에 적용할 수 있는 템플릿입니다.
필요에 맞게 수정하여 사용하세요.
