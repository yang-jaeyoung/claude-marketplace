# Templates Directory

Project templates are located in the knowledge base.

## Actual Location

All templates are in: **`../knowledge/templates/`**

### Structure:
```
knowledge/templates/
├── ci/            # CI/CD templates (GitHub Actions, etc.)
├── config/        # Configuration file templates
├── generators/    # Rails generator templates
└── project/       # Full project templates
```

## Available Template Categories

1. **CI/CD** (`knowledge/templates/ci/`)
   - GitHub Actions workflows
   - GitLab CI configurations
   - Testing pipelines

2. **Configuration** (`knowledge/templates/config/`)
   - Database configurations
   - Environment settings
   - Service configurations

3. **Generators** (`knowledge/templates/generators/`)
   - Custom Rails generators
   - Template files

4. **Project** (`knowledge/templates/project/`)
   - Full Rails application templates
   - Starter projects
   - Boilerplate code
